# ❓ أسئلة شائعة | FAQ

## 🎯 أسئلة عامة | General Questions

### س: ما هو تطبيق NubaFarm؟
**ج:** تطبيق إدارة المزارع الحديثة يساعدك على:
- إدارة الإنتاج اليومي
- تتبع المخزون
- إدارة الشحنات
- طباعة الملصقات والباركود

### س: ما الفرق بين الموقع والتطبيق؟
**ج:** لا يوجد فرق تقريباً! التطبيق هو نفس الموقع بالضبط لكن:
- ✓ يعمل على الهاتف
- ✓ أسرع وأخف وزناً
- ✓ يمكن استخدامه بدون إنترنت (بعد التحميل الأول)
- ✓ تحديثات تلقائية

### س: هل يمكن استخدام البيانات بين الموقع والتطبيق؟
**ج:** نعم، جميع البيانات تُحفظ محلياً ويمكن مزامنتها بسهولة.

## 📱 أسئلة التثبيت | Installation Questions

### س: ما المتطلبات لتثبيت التطبيق؟
**ج:** 
- Android 5.0 أو أحدث
- مساحة حرة 100 MB على الأقل
- ذاكرة 2 GB على الأقل (الموصى به 4 GB)

### س: كم حجم التطبيق؟
**ج:** حوالي 15-20 MB فقط (خفيف جداً)

### س: هل يمكن تثبيت التطبيق على جهازين؟
**ج:** نعم، لا توجد مشكلة. فقط استخدم نفس البيانات في كليهما.

### س: كيف أحذف التطبيق؟
**ج:**
```
Settings → Apps → NubaFarm → Uninstall
```

### س: هل يمكن تثبيت التطبيق على Tablet؟
**ج:** نعم، التطبيق يعمل على جميع الأجهزة بشاشات مختلفة.

## ⚙️ أسئلة تقنية | Technical Questions

### س: هل التطبيق آمن؟
**ج:** نعم جداً:
- ✓ البيانات محفوظة محلياً على جهازك
- ✓ لا تُرسل بيانات حساسة
- ✓ الكود محمي (ProGuard)
- ✓ دعم أخر إصدار Android

### س: هل يحتاج الإنترنت؟
**ج:** 
- ✓ بعد التثبيت الأول، يعمل بدون إنترنت (وضع محلي)
- ✓ الإنترنت مطلوب فقط للتحديثات

### س: ماذا يحدث للبيانات عند تحديث التطبيق؟
**ج:** جميع بيانات اللوكال ستوريج تُحفظ بأمان:
- ✓ لا حاجة للقلق
- ✓ بياناتك آمنة 100%

### س: هل يمكن نقل البيانات إلى جهاز آخر؟
**ج:** نعم، استخدم وظيفة النسخ الاحتياطي:
```bash
adb backup -apk -all -f backup.ab
adb restore backup.ab
```

### س: كيف أرى السجلات/الأخطاء؟
**ج:** للمطورين:
```bash
adb logcat | grep nubafarm
```

## 🖨️ أسئلة الطباعة | Printing Questions

### س: كيف أطبع الملصقات من التطبيق؟
**ج:**
1. اذهب إلى "الإنتاج" → اختر منتج
2. اضغط "طباعة" أو "معاينة"
3. استخدم طابعة حرارية
4. اضغط "طباعة"

### س: ما نوع الطابعة المدعوم؟
**ج:** 
- ✓ طابعات حرارية (مثل Zebra)
- ✓ طابعات عادية
- ✓ طابعات شبكة
- ✓ الطابعات الافتراضية للنظام

### س: لماذا الملصقات غير واضحة؟
**ج:** تحقق من:
- [ ] جودة الورق
- [ ] رأس الطباعة نظيف؟
- [ ] الإعدادات في التطبيق صحيحة
- [ ] التحديث إلى أحدث نسخة

### س: هل يدعم طباعة عدة ملصقات؟
**ج:** نعم:
- ✓ حدد عدة منتجات
- ✓ اضغط "طباعة جماعية"
- ✓ كل الملصقات ستُطبع

### س: كيف أغير حجم الملصق؟
**ج:** 
```
الإعدادات → الطباعة → حجم الملصق
- صغير (50x100mm)
- متوسط (100x150mm)
- كبير (150x200mm)
```

## 💾 أسئلة البيانات | Data Questions

### س: كم عدد المنتجات التي يمكن حفظها؟
**ج:** بدون حد تقريباً (تعتمد على مساحة الهاتف)

### س: هل البيانات تُحذف عند إعادة تشغيل؟
**ج:** لا، البيانات محفوظة بشكل دائم في Local Storage

### س: كيف أمسح جميع البيانات؟
**ج:**
```
Settings → Apps → NubaFarm → Storage → Clear All
```
⚠️ تحذير: هذا سيحذف جميع بيانات التطبيق!

### س: هل يمكن استيراد البيانات من ملف؟
**ج:** نعم، تحت الإعدادات:
- ✓ استيراد من ملف CSV
- ✓ استيراد من ملف Excel
- ✓ استيراد من JSON

### س: كيف أُنسخ البيانات إلى USB؟
**ج:**
```bash
adb pull /sdcard/NubaFarm/data.json ~/Desktop/
```

## 🚀 أسئلة الأداء | Performance Questions

### س: لماذا التطبيق بطيء أحياناً؟
**ج:** تحقق من:
- [ ] هل الجهاز لديه مساحة حرة؟
- [ ] هل هناك تطبيقات أخرى مفتوحة؟
- [ ] جرّب إعادة تشغيل الجهاز

### س: كيف أحسّن السرعة؟
**ج:**
```
1. امسح ذاكرة التخزين المؤقتة
Settings → Apps → NubaFarm → Storage → Clear Cache

2. أغلق التطبيقات الأخرى

3. جرّب الوضع الآمن (Safe Mode)
```

### س: كم مساحة تحتاج البيانات؟
**ج:** حسب عدد المنتجات:
- ألف منتج = ~1-2 MB
- عشرة آلاف = ~10-20 MB
- مليون = ~100-200 MB
