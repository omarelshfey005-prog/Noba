# 🚀 البدء السريع | Quick Start Guide

## 5 دقائق للبدء! | Get Started in 5 Minutes

## 📋 المتطلبات الأساسية | Prerequisites

```
✓ Android 5.0 أو أحدث
✓ 100 MB مساحة حرة
✓ اتصال إنترنت (للتحميل الأول فقط)
```

## 1️⃣ التثبيت | Installation

### للمستخدمين العاديين:

#### الطريقة الأولى: Google Play Store
```
قريباً! اتبع التحديثات
```

#### الطريقة الثانية: ملف APK
```
1. انسخ app-debug.apk إلى هاتفك
2. افتح File Manager
3. اضغط على الملف
4. اضغط Install
5. اسمح بالأذونات
```

#### الطريقة الثالثة: ADB (للمطورين)
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 2️⃣ الفتح الأول | First Launch

عند فتح التطبيق لأول مرة:

```
1. رسالة الترحيب

   ⏳ جاري تحميل البيانات...

4. لوحة التحكم
   ✓ جاهز للاستخدام!
```

## 3️⃣ الاستخدام الأساسي | Basic Usage

### 📊 لوحة التحكم (Dashboard)
```
الشاشة الرئيسية تعرض:
• عدد المنتجات اليومي
• عدد الشحنات
• إجمالي الإنتاج
• إحصائيات سريعة
```

### 📦 الإنتاج (Production)
```
1. اضغط "إضافة منتج جديد"
2. أدخل البيانات:
   - اسم المنتج
   - الكمية
   - البيانات الإضافية
3. اضغط "حفظ"
4. (اختياري) اضغط "طباعة"
```

### 📋 المخزون (Inventory)
```
1. شاهد جميع المنتجات
2. ابحث عن منتج معين
3. عدّل المنتج
4. احذف إذا لزم الأمر
```

### 🚢 الشحنات (Shipments)
```
1. اضغط "شحنة جديدة"
2. اختر المنتجات
3. أدخل التفاصيل
4. اضغط "إرسال"
```

### ⚙️ الإعدادات (Settings)
```
• اللغة
• الأمان والكلمة المرورية
• إعدادات الطباعة
• مسح البيانات
• الدعم والمساعدة
```

---

## 🖨️ الطباعة | Printing

### طباعة ملصق واحد:
```
1. اختر منتج من الإنتاج
2. اضغط الزر "🖨️ طباعة"
3. اختر الطابعة من القائمة
4. اضغط "طباعة" في الحوار
```

### طباعة عدة ملصقات:
```
1. في المخزون، حدد عدة منتجات (✓)
2. اضغط "طباعة جماعية"
3. أدخل عدد النسخ
4. اضغط "طباعة"
```

### معاينة الملصق:
```
1. اضغط 👁️ (عين) بدل الطباعة
2. شاهد كيف سيبدو الملصق
3. عدّل إذا لزم
4. اطبع عندما تكون جاهز
```

## 💾 الحفظ والبيانات | Saving & Data

### تحفظ البيانات تلقائياً ✓

**عندما تحفظ بيانات:**
- ✓ تُحفظ في Local Storage
- ✓ آمنة 100%
- ✓ تبقى حتى إذا أغلقت التطبيق
- ✓ تبقى حتى لو أعدت التشغيل

---

## ⚙️ الإعدادات المهمة | Important Settings

### تفعيل جميع الأذونات:
```
الإعدادات → Apps → NubaFarm → Permissions
✓ Storage
✓ Location (اختياري)
✓ Internet
```

### تغيير كلمة المرور:
```
Settings → أمان → تغيير كلمة المرور
1. أدخل القديمة
2. أدخل الجديدة
3. أكد الجديدة
4. اضغط "تحديث"
```

### استيراد البيانات:
```
Settings → بيانات → استيراد
1. اختر ملف (CSV, Excel, JSON)
2. اختر نقاط البيانات
3. اضغط "استيراد"
```

## 🔍 البحث | Search

### البحث السريع:
```
في أي قائمة:
1. اضغط على شريط البحث 🔍
2. اكتب اسم المنتج
3. سيظهر الناتج فوراً
4. اضغط على النتيجة
```

### الفلترة (Filtering):
```
في المخزون:
1. اضغط على "المرشحات"
2. اختر المعايير:
   • النوع
   • التاريخ
   • الحالة
3. اضغط "تطبيق"
```

## 📊 التقارير | Reports

### عرض التقارير:
```
Dashboard → Reports
يعرض:
• إجمالي الإنتاج اليومي
• الشحنات هذا الأسبوع
• أكثر منتج مبيعاً
• الإحصائيات الشهرية
```

### تصدير التقرير:
```
1. من التقرير، اضغط "📤 تصدير"
2. اختر الصيغة:
   • PDF
   • Excel
   • CSV
3. اختر مكان الحفظ
```

## 🆘 استكشاف الأخطاء | Troubleshooting

### التطبيق لا يفتح:
```
1. أعد تشغيل الهاتف
2. امسح ذاكرة التخزين
3. أعد تثبيت التطبيق
```

### البيانات لا تحفظ:
```
تحقق من:
✓ وجود مساحة حرة كافية
✓ الأذونات مفعلة
✓ لا يوجد تطبيق آخر يستخدم نفس الملفات
```

### الطباعة لا تعمل:
```
1. تأكد من توصيل الطابعة
2. جرّب طابعة أخرى
3. أعد تشغيل الطابعة
4. تحقق من الاتصال بالشبكة
```
